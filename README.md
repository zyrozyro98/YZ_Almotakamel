# YZ_Almotakamel - المنصة المتكاملة لإدارة الطلاب وخدمة العملاء

هذا المشروع هو حل برمجى متكامل (Full-stack) لإدارة عمليات تسجيل الطلاب، مراقبة الموظفين، والتواصل المباشر عبر الواتساب. مصمم خصيصاً ليناسب متطلبات العمل الاحترافية مع واجهة مستخدم متطورة تدعم اللغة العربية بالكامل (RTL).

## 🚀 المميزات الرئيسية
- **لوحة تحكم (Dashboard) ذكية:** إحصائيات فورية وتتبع للأداء.
- **إدارة الطلاب والطلبات:** نظام متكامل لإضافة وتعديل حالات الطلاب مع أتمتة توزيع المهام.
- **دردشة واتساب مدمجة:** تواصل مباشر مع العملاء من داخل المنصة مع دعم الرسائل الصوتية والوسائط.
- **نظام التوزيع الذكي (Smart Distribution):** توزيع تلقائي للطلبات الجديدة على الموظفين حسب ضغط العمل.
- **تقارير ورقابة (Admin Control):** تصدير البيانات إلى Excel ومراقبة حالة اتصال الموظفين لحظياً.
- **تصميم متجاوب (Responsive Design):** تجربة مستخدم مثالية على أجهزة الحاسوب، الأجهزة اللوحية، والهواتف الذكية.

## 🛠 التكنولوجيا المستخدمة
- **Frontend:** React.js, Vite, Lucide React, Axios.
- **Backend:** Node.js, Express, Baileys (WhatsApp Web API).
- **Database:** Firebase Firestore & Realtime Database.
- **Styling:** Modern Vanilla CSS (Glassmorphism UI).

## 💻 طريقة التشغيل

### المتطلبات الأساسية
- Node.js (v18+)
- npm

### الخطوات
1. **تحميل المشروع:**
   ```bash
   git clone [repository-url]
   cd YZ_Almotakamel
   ```

2. **تثبيت المكتبات:**
   ```bash
   npm run install:all
   ```

3. **إعداد البيئة:**
   - قم بتعديل ملفات `.env` في كل من مجلد `frontend` و `backend` لإضافة مفاتيح Firebase الخاصة بك.

4. **التشغيل التجريبي (Development):**
   ```bash
   npm run dev
   ```

5. **البناء للنشر (Production):**
   ```bash
   npm run build
   ```

## 🐳 التشغيل عبر Docker
يمكنك تشغيل المنصة بالكامل (Frontend + Backend) باستخدام Docker:
```bash
docker-compose up --build
```
